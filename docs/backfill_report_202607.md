## 方法（v2，修正 v1 两个致命假设）

1. Wayback CDX 前缀查询枚举外交部"重要新闻"三代栏目 URL（`mfa_chn/zyxw_602251` → `fmprc/web/zyxw` → `mfa/web/zyxw`），去重得 10,338 篇唯一文章快照；
2. `web/{ts}id_/` 原始模式逐篇抓正文（3 并发实测最优，>4 触发按 IP 限流退避雪崩），9,374 篇成功、16 篇永久 404（0.17%）；
3. 标题筛选（主持人扩展 李克强/胡锦涛/温家宝）→ 正文地点判别 → 视频峰会剔除 → 领导人+10 天窗去重。

v1 缺陷：老站文章 URL 是流水号 `t\d+.shtml` 而非 `t20\d{6}` 日期式（会颗粒无收）；老文章现网已成"系统维护"壳页，正文只能取快照。

## 缺口原因（按影响排序）

1. **Wayback 捕获宇宙有限**：外交部年发文约 3,000 篇，Wayback 仅捕获 100–1,500 篇/年；2013–2014 与 2018 最稀。
2. **峰会周轮换过快**：2018 FOCAC（约 50 国元首）、2014 APEC 等，通稿一天十几篇，首页快照跟不上；纯参会未获双边会见的领导人本来就无通稿（README 已知事项 2）。
3. 已排查且否决的补充源：外交部"领导人活动"专栏（Wayback 0 捕获）、gov.cn（内容页捕获密集但现网已 404、列表页快照仅 48 个/11 年）。

## 峰会名单信源优先级（回填手册·中国主场峰会通用）

> FOCAC 2024 实战验证的方法链（master 中峰会周 46 条与欢迎宴会"51 国+2 位总统代表"口径对齐，差额为仅参会未获双边者）：

1. **峰会专题官网档案**（`2024focacsummit.mfa.gov.cn` 类）——"最新要闻"栏**不滚动**，是完整档案（FOCAC 2024: /kms/ 16 页 219 篇）；从双边会见稿标题批量提取（注意主方不止习近平：李强/赵乐际/丁薛祥/李希，头衔含 总统委员会主席/主权委员会主席，国名含括号如 刚果（布））。
2. **欢迎宴会通稿**——给总数口径（"51 国元首/政府首脑+2 位总统代表"），用于差额说明。
3. **峰会周双边会见稿批量提取**（zyxw 语料）——与 1 高度重叠，互为校验。
4. **中文维基条目**——有错误率（FOCAC 2018 坦桑尼亚代表写错），须与语料互证。
5. **央视"抵京"系列**——逐人核对差额时用。

适用清单（专题官网大多在线）：2018 FOCAC、2017/2019 一带一路、2014 APEC、2016 G20 杭州、2022 冬奥、2023 中国-中亚西安峰会、2025 上合天津。

## 剩余缺口与下一步（新弧）

剩余 ~165 次缺口分布：2013(-51)、2014(-24)、2018(-27)、2024(-19)、2015(-18) 为主。

- **2013/2014/2015**：无在华巨型峰会可补，纯粹是 Wayback 早期捕获稀疏；出路是英文版外交部
  （`/eng/`，快照独立于中文）或 gov.cn/xinwen 定向窗口抓取（仅 2018-09 就有 2,735 篇内容页快照）。
- **2018 (-27)**：首届进博会（2018-11）十余位元首——可按上面方法链走进博会专题官网。

- 已试无果：外交部"领导人活动"专栏（Wayback 0 捕获）、gov.cn 列表页快照（11 年仅 48 个）、
  en.wiki FOCAC 2024（404）、进博会/FOCAC2024 中文维基（无名单/无专页）。
