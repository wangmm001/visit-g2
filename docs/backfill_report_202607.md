## 方法（v2，修正 v1 两个致命假设）

1. Wayback CDX 前缀查询枚举外交部"重要新闻"三代栏目 URL（`mfa_chn/zyxw_602251` → `fmprc/web/zyxw` → `mfa/web/zyxw`），去重得 10,338 篇唯一文章快照；
2. `web/{ts}id_/` 原始模式逐篇抓正文（3 并发实测最优，>4 触发按 IP 限流退避雪崩），9,374 篇成功、16 篇永久 404（0.17%）；
3. 标题筛选（主持人扩展 李克强/胡锦涛/温家宝）→ 正文地点判别 → 视频峰会剔除 → 领导人+10 天窗去重。

v1 缺陷：老站文章 URL 是流水号 `t\d+.shtml` 而非 `t20\d{6}` 日期式（会颗粒无收）；老文章现网已成"系统维护"壳页，正文只能取快照。

## 缺口原因（按影响排序）

1. **Wayback 捕获宇宙有限**：外交部年发文约 3,000 篇，Wayback 仅捕获 100–1,500 篇/年；2013–2014 与 2018 最稀。
2. **峰会周轮换过快**：2018 FOCAC（约 50 国元首）、2014 APEC 等，通稿一天十几篇，首页快照跟不上；纯参会未获双边会见的领导人本来就无通稿（README 已知事项 2）。
3. 已排查且否决的补充源：外交部"领导人活动"专栏（Wayback 0 捕获）、gov.cn（内容页捕获密集但现网已 404、列表页快照仅 48 个/11 年）。
